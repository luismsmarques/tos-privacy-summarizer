1. Aceitação dos Termos

Ao instalar e utilizar a extensão "ToS & Privacy Summarizer" ("a extensão", "o serviço"), você concorda em cumprir e estar vinculado a estes Termos de Serviço. Se não concordar com qualquer parte destes termos, não deve utilizar a extensão.

📋 Resumo dos Termos

Estes termos regem o uso da extensão que utiliza IA para resumir documentos legais. Ao utilizar o serviço, você concorda com a recolha limitada de dados, o uso da API Gemini da Google, e as limitações de responsabilidade aqui descritas.

2. Descrição do Serviço

2.1 Funcionalidade Principal

  • Análise de documentos legais: A extensão analisa Termos de Serviço e Políticas de Privacidade
  • Resumos com IA: Utiliza a API Gemini da Google para gerar resumos estruturados
  • Interface intuitiva: Apresenta informações de forma clara e acessível
  • Gestão de créditos: Sistema de créditos para uso da API compartilhada

2.2 Limitações do Serviço

  • Precisão: Os resumos são gerados por IA e podem conter imprecisões
  • Completude: Os resumos não substituem a leitura completa dos documentos originais
  • Atualização: Os resumos podem não refletir alterações recentes nos documentos
  • Disponibilidade: O serviço depende da disponibilidade de terceiros (Google Gemini API)

3. Uso Aceitável

3.1 Uso Permitido

  • Uso pessoal: Para análise pessoal de documentos legais
  • Uso educacional: Para fins de aprendizagem e pesquisa
  • Uso profissional: Para análise de documentos no contexto profissional

3.2 Uso Proibido

  • Uso comercial: Não pode utilizar para serviços comerciais sem autorização
  • Distribuição: Não pode redistribuir ou revender a extensão
  • Engenharia reversa: Não pode tentar descompilar ou modificar o código
  • Uso malicioso: Não pode utilizar para atividades ilegais ou prejudiciais

⚠️ Aviso Importante

Os resumos gerados são apenas para fins informativos. Não constituem aconselhamento jurídico e não devem ser utilizados como base para decisões legais importantes. Sempre consulte um advogado qualificado para questões jurídicas específicas.
